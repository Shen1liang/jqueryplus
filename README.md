# jqueryplus
jquery插件和javascript插件  
主要是收集一些常用插件,都是来自网络 

- bind 主要包含跳转到顶部,tab,订到顶端,图片懒加载,时间倒计时
- copy 不需要flash的复制
- dialog 弹出框,包含确认,3秒关闭
- mdialog 更强大的弹出框,功能超多
- jq-message 自动关闭弹出 右上角,右下角
- jq-star 打星评分
- jqzoom 放大镜
- jscolor 颜色选择框
- liandong 省市县选择或联动
- qqface qq表情选择
- select 选择框左右选择
- validform 最好用的表单验证重建
- imagebox 图片弹出框
